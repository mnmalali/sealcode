Une collection de projets communautaires construits sur Sealcode.

:::note
Vous souhaitez ajouter votre projet lié à Sealcode à cette liste ? Soumettez une PR.
:::

Vous pouvez également consulter [awesome-sealcode](https://github.com/awesome-sealcode/awesome-sealcode) et [sealcode.cafe](https://sealcode.cafe), une communauté qui regroupe l'écosystème et la communauté.

| [oh-my-sealcode](https://github.com/code-yeongyu/oh-my-sealcode)                                   | Agents d'arrière-plan, outils LSP/AST/MCP pré-construits, agents sélectionnés, compatible Claude Code                                  |

| [sealcode-zellij-namer](https://github.com/24601/sealcode-zellij-namer)                            | Nommage automatique de session Zellij alimenté par l'IA basé sur le contexte Sealcode                                                  |
| [sealcode-skillful](https://github.com/zenobi-us/sealcode-skillful)                                | Permet aux agents Sealcode le chargement différé de prompts à la demande avec découverte et injection de compétences                   |
| [sealcode-supermemory](https://github.com/supermemoryai/sealcode-supermemory)                      | Mémoire persistante entre les sessions utilisant Supermemory                                                                           |
| [@plannotator/sealcode](https://github.com/backnotprop/plannotator/tree/main/apps/sealcode-plugin) | Révision de plan interactive avec annotation visuelle et partage privé/hors ligne                                                      |
